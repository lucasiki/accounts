Como usar.

acessar static/textdb.xlsx e configurar os textos

Utilizar register.html para criar conta, modifique o css como quiser

Utilizar login.html para acessar conta, modifique o css como quiser

Utilizar manage.html para gerenciar contas, modifique o css como quiser

a pagina loggedin é só um template para o login, pode altera-lo ou fazer com que redirecione para a página que deseja.

a página logoff faz logof, trate no views como quiser.
